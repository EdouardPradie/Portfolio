# Portfolio - Edouard Pradie

My personal portfolio showcasing my experience, skills and projects - now built with Vue.js!

## Contents

- **index.html** - Main Vue.js portfolio page with:
  - Personal introduction
  - Professional experience
  - Education
  - Technical skills
  - GitHub projects (loaded dynamically)
  - CAMI Final Study Project section
  - Interests

- **cv.html** - Resume in ATS format (optimized for applicant tracking systems)
- **cv_en.html** - English version CV
- **cv_fr.html** - French version CV

## Technologies

- Vue.js 3
- Vite (build tool)
- HTML5 / CSS3
- Responsive design (mobile-friendly)
- GitHub API to dynamically load projects

## Getting Started

### Prerequisites
- Node.js 18+ and npm

### Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# The app will be available at http://localhost:5173
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview

# Output will be in the dist/ directory
```
